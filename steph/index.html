Pricing

Prices will vary a lot depending on what you're interested in.

Both the amount/type of yarn used in a project and the amount of time it takes me to make will factor into how much I charge. Hats usually run $30 - $35, scarves can be between $30 and $55, coozies are $15. Bigger projects like blankets will be more expensive and vary more as well. If you have any questions, please ask!
